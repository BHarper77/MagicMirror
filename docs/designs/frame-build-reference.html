A
Matte black, low sheen. A glossy frame throws reflections that fight the mirror. Black-stained pine/poplar or black aluminium frame both work; keep the finish flat.
B
Keep the display background black. Two-way mirror only shows lit pixels — the MagicMirror layout is already black, so the reveal and unused screen read as mirror.
C
Two 3-pin sockets inside. Monitor 19V adapter + Pi official 5V USB-C on a certified BS 1363 trailing block, sideways in the bottom rail. Leave airflow — it runs 24/7. — ADR-0001
D
Confirm auto-power-on before sealing. Verify the monitor wakes when mains is applied, or a power cut leaves a black mirror with no way to switch it on.
